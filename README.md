# MQWAY

**MQ Way로 키우는 경제 지능·자기주도력 통합 교육 플랫폼**

## 프로젝트 소개

MQWAY는 어린이와 청소년이 MQ Way 방식으로 경제 지능과 자기주도력을 함께 키울 수 있는 통합 교육 플랫폼입니다.

### 주요 기능

- 📰 **경제 뉴스**: 어린이를 위한 맞춤형 경제 뉴스 제공
- 🎯 **투자 포트폴리오**: 투자 학습을 위한 포트폴리오 관리
- 🧠 **경제 교육 게임**: 재미있는 게임을 통한 경제 개념 학습
- 📊 **섹터 분석**: 다양한 산업 섹터에 대한 분석 정보
- 💰 **캐시플로우 게임**: 현금 흐름 관리 학습 게임

### 기술 스택

- **Backend**: Laravel (PHP)
- **Frontend**: Blade Templates, JavaScript
- **Database**: MySQL
- **Infrastructure**: Docker, Nginx
